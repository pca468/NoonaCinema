# 맨 처음 프로젝트 시작할 시에는 
터미널에서 git clone https://github.com/pca468/NoonaCinema.git
입력하여 시작해줍니다.

그 다음부터는 과제에서 했던 것과 동일하게
git init //
git add .  //
git commit -m "" //
git remote 레퍼지토리 주소 (처음 push 한번만 진행 이후 생략가능) //
git push origin master  //

순으로 github에 올려주시면 됩니다.
